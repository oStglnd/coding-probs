# arrayRotate

## Problem description

Write a function that takes as input one array and a positive integer `k`. Then, the function should "rotate" the array for the number of steps corresponding to `k`. For example, if the input is `[1, 2, 3, 4, 5]`, `k=2`, the output should be `[4, 5, 1, 2, 3]`.
